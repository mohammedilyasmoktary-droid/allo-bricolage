# Deploying Database Schema to Hostinger

## Option 1: Run Prisma Commands via SSH (Recommended)

If you have SSH access to your Hostinger server:

1. **SSH into your Hostinger server:**
```bash
ssh your-username@your-hostinger-server
```

2. **Upload your project files** (or clone from Git if you have a repository)

3. **On the server, navigate to your backend directory:**
```bash
cd /path/to/your/backend
```

4. **Update .env with localhost:**
```env
DATABASE_URL="mysql://u905810677_adminbrico:ALObricolage11@localhost:3306/u905810677_alobricolage"
```

5. **Run Prisma commands:**
```bash
npm install
npx prisma generate
npx prisma db push
```

## Option 3: Use phpMyAdmin to Import Schema

You can also manually create the tables using phpMyAdmin:

1. Go to phpMyAdmin
2. Select your database `u905810677_alobricolage`
3. Click on **"SQL"** tab
4. Run the SQL commands generated by Prisma

To generate SQL from Prisma:
```bash
npx prisma migrate dev --create-only --name init
```

This creates migration files you can convert to SQL.

# Allo Bricolage - Moroccan Maintenance Marketplace

A full-stack web application connecting clients with skilled technicians for home maintenance and repair services in Morocco.

## Tech Stack

### Frontend
- React + TypeScript
- Vite
- Material-UI
- React Router

### Backend
- Node.js + Express
- TypeScript
- Prisma ORM
- MySQL (Hostinger)
- JWT Authentication

## Features

- User authentication (Clients & Technicians)
- Service booking system
- Real-time technician search
- Payment processing
- Review and rating system
- Subscription management for technicians
- Admin dashboard
- Profile management with photo uploads

## Setup

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm run dev
```

## Environment Variables

See `.env.example` files in frontend and backend directories.

## Database

MySQL database hosted on Hostinger.

## Deployment

- Frontend: Hostinger `public_html/`
- Backend: Hostinger Node.js hosting or VPS
- Database: Hostinger MySQL

## Image Requirements

- Recommended size: 1200x800px or similar aspect ratio
- Format: JPG, PNG, or WebP
- File size: Optimized for web (under 500KB per image recommended)

## Usage

These images are used in:
1. **Hero Carousel** - Rotating banner at the top of the homepage
2. **Technician Showcase** - Grid display of featured technicians
3. **Service Categories** - Background images for service cards

The application will automatically handle image loading and provide fallbacks if images are missing.
